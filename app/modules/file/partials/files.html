What's Forge?

Forge is a tool created to edit client data for Five Clans Online. Forge allows you to edit item tooltip descriptions, item set descriptions, add display strings, and update the file manifest for loading game assets. It was created by the FCO team to ease the painstaking process of patch development.

To get started, just load in the necessary client files below and then select the appropriate tab above.
